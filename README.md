# BAZAR — маркетплейс (макет)

Кликабельный прототип торговой площадки типа Авито для Кыргызстана.
Чистый HTML/CSS/JS, без сборки и бэкенда — состояние в localStorage.

**Возможности:** умный поиск с NLU («айфон до 50к бу в бишкеке»), ИИ-помощница Айка,
SVG-карта КР, фильтры, чаты, подача объявлений, индикатор цены vs рынок,
3 языка (RU/EN/KY), темы (светлая/тёмная/системная), адаптив под мобильные.

Это демо-макет на мок-данных (~580 объявлений), не прод.
